🦊 Foxbook

Verifiable agent identity primitive for A2A and MCP.

Stable

An RFC 9162-shaped public transparency log. verifyAgentCard(card) returns one of four outcomes — verified, unverified, handle-mismatch, stale-proof — backed by signed Merkle inclusion proofs. Apache 2.0. Run your own log; anyone following the spec can verify against any deployment.

curl -s https://transparency.foxbook.dev/root | jq
